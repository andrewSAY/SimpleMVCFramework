Привет. Добро пожаловать на бесплатный хостинг 500mb.net

Если ты видишь этот текст, значит скорее всего именно ты зарегистрировал этот домен на бесплатном хостинге 500mb.net и уже готов разместить здесь свой сайт!

Для этого тебе нужно совсем немного, либо зайти в панель управления своим хостингом cPanel и выбрать пункт Online File Manager, далее зайти в каталог htdocs и загрузить файлы с компьютера, либо закачать содержимое через ftp (лучше всего для этого использовать FileZilla Client).

Сайт будет запускаться с файлов: index.html или index.php находящихся в корне папки htdocs.

Как установить CMS и прикрепить свой домен на нашем бесплатном хостинге смотри наш раздел FAQ.
